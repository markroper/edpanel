Read and Write Access

Introduction

Read and write access to PowerSchool and PowerTeacher gradebook data is supported through a set of web services, which follow the Representational State Transfer (REST) services architecture.

These services offer a number of supported operations, such as retrieving all students in a school, retrieving a student by ID, etc. For a listing of all operations that are available and how to use them, see Available Resources. These operations share common features, such as pagination, common syntax for expanding upon data groups, and searching at the school or district office level. For details on how to use common features and syntax, see General Usage.

Data input and output for these services are defined by a data model and includes objects, such as school, student, student school enrollment, etc. For a listing and definition of these objects, see Data Dictionary.
