PowerSchool REST API

Introduction

The PowerSchool API supports a range of data access options to meet the integration needs of your application, including:

  • Read Access so that you can provision data, search for and page data from PowerSchool, and retrieve assignment data from PowerTeacher gradebook, all in real time.
  • Write Access so that you can create and score assignments in PowerTeacher gradebook in real time.

Due to security reasons, data access is provided only to authenticated applications.

You can also build a lightweight webhook to receive data change notifications from PowerSchool.
