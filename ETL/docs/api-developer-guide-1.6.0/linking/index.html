Linking

Introduction

Linking allows PowerSchool users to quickly and easily access your application from PowerSchool.

This document assumes a general knowledge of XML document structure and syntax.

Contents

Declare Navigation Links
Adding navigation links to the PowerSchool user interface.

Declare Navigation Links

Adding navigation links to the PowerSchool user interface is very easy. You simply need to supply a <links> element to the <plugin> tag.

This sample XML defines a plugin for MyApp with two separate link definitions, each with their own locations on the user interface specified. These links will appear in PowerSchool and allow the user to navigate to the MyApp application. The first link will appear in both the navigation toolbar and the main menu (left navigation) of the PowerSchool admin portal. The second link will only appear in the navigation toolbar of the PowerSchool admin portal. 

<?xml version="1.0" encoding="UTF-8"?>
<plugin xmlns="http://plugin.powerschool.pearson.com"
        xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
        xsi:schemaLocation='http://plugin.powerschool.pearson.com plugin.xsd'
        name="MyApp"
        version="1.2.3"
        description="Link to my third-party application">

  <links>
    <link title="MyApp Login" display-text="MyApp" path="http://mycompany.com:443/login.aspx">
      <ui_contexts>
        <ui_context id="admin.header" />
        <ui_context id="admin.left_nav" />
      </ui_contexts>
    </link>
    <link title="MyApp Information" display-text="MyApp Info" path="http://mycompany.com:443/info.aspx">
      <ui_contexts>
        <ui_context id="admin.header" />
      </ui_contexts>
    </link>
  </links>

  <publisher name="Example Publisher">
    <contact email="publisher@example.com" />
  </publisher>

</plugin>
